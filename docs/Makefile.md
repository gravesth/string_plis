Статья по составлению makefile 
Индентация осуществляется исключительно при помощи символов табуляции, # каждой команде должен предшествовать отступ 
```
<цели>: <реквизиты> 
	<команда #1>
	 ... 
	 <команда # 2>
```

То есть, правило make это ответы на три вопроса:
```
{Из чего делаем? (реквизиты)} ---> [Как делаем? (команды)] ---> {Что делаем? (цели)}
```

Несложно заметить что процессы трансляции и компиляции очень красиво ложатся на эту схему:  
```
{исходные файлы} ---> [трансляция] ---> {объектные файлы}
{объектные файлы} ---> [линковка] ---> {исполнимые файлы}
```
